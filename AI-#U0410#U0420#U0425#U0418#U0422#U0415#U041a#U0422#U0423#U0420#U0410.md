# RYNOW AI

## Ограниченная база знаний

Модель получает только найденные фрагменты из `data/<server>-rules.json`.

В API не включён xAI Web Search. Поэтому модель технически не получает
доступ к веб-поиску в рамках запроса сайта.

## Защита от выдуманных ответов

Системная инструкция требует:

- не использовать внешние знания;
- не придумывать пункты;
- сообщать об отсутствии данных;
- указывать использованные источники.

Полностью исключить ошибки языковой модели невозможно, поэтому интерфейс
всегда показывает оригинальные источники для проверки.
