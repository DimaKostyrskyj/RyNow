# RYNOW — профиль, история, избранное и уровни

## Логотипы проектов

Положите свои PNG-файлы в:

```text
assets/projects/
```

Названия должны быть:

```text
project-1.png
project-2.png
project-3.png
project-4.png
project-5.png
```

SVG-заглушки можно оставить: браузер автоматически использует PNG, когда он существует.

## Discord

На сайте используется локальный файл:

```text
assets/discord-logo.svg
```

Это настоящий знак Discord, а не текстовый символ.

## Профиль

После входа через Discord показываются:

- аватар Discord;
- имя Discord;
- Discord ID;
- электронная почта, если Discord её вернул;
- роль «Основатель» для двух указанных ID;
- уровень и XP;
- история вопросов;
- избранные ответы;
- настройки.

## Уровни

Начальный уровень: 1.

XP начисляется:

- +10 XP за первый вход в течение дня;
- +20 XP за вопрос AI;
- +5 XP за добавление ответа в избранное.

## Хранение данных

История, избранное, настройки и XP сохраняются в `localStorage` отдельно для каждого Discord ID.

Это значит:

- данные сохраняются после перезагрузки;
- работают без отдельной базы данных;
- данные привязаны к браузеру и устройству.

Для синхронизации между компьютером и телефоном потребуется база данных.
